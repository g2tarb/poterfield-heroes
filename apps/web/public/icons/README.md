# Icons placeholder

Le DA produira les icônes finales (cf. Section 13 du brief DA UX).

En attendant, génère des placeholders carrés en noir #0A0A0B avec un mot "PH" (ex: realfavicongenerator.net ou Figma → export).

Fichiers attendus :

- `icon-192.png` — 192×192 (Android home)
- `icon-512.png` — 512×512 (Android splash)
- `icon-maskable-512.png` — 512×512 avec padding sécurité (mascable Android)
- `apple-touch-icon.png` — 180×180 (iOS home)
- `favicon.ico` — 32×32 multi-format

Le `manifest.webmanifest` les référence déjà : tu déposes les fichiers ici et c'est lu automatiquement.
